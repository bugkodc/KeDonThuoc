# soud code kê đơn thuốc
## cài đặt 
```
B1 : clone code về từ github
B2 : vào unity chọn add , chọn folder chứa code mới clone 
B3 : copy folder Unity-CNPM vào mục htdocs của xampp (C:\xampp\htdocs)
B4 : chạy project mới add vào trong unity
B5 : mở xampp và chạy http://localhost/phpmyadmin/
B6 : tạo database có tên là "donthuoc" và up file donthuoc.sql lên
```
